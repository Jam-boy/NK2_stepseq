■ midiglue
in      : USB Host(KORG nano KONTROL2)
output1 : CV out
output2 : GATE out

Knob L            : BPM set
Knob R            : Tdiv set
Button A + Knob L : Swing set
Button A + Knob R : NoteRange set
Button B + Knob L : Key set
Button B + Knob R : Scale set


■ KORG nano KONTROL2
Knob 1-8        : Portament set
Slider 1-8      : NoteNumber set
Solo Button 1-8 : The corresponding step will sound while you press it.
Mute Button 1-8 : The lit steps will sound. (Default is off)
Track Rew       : Reduce the number of steps used.
Track FF        : Increase the number of steps used.

CYCLE           : Random
Rew             : Down
FF              : Up
Rew + FF        : Up + Down

STOP            : Stop
PLAY            : Play
REC             : Scale set ON/OFF

Set             : Change portamento settings（Knob1 / Knob1-8）
Marker Rew      : -1 Oct
Marker FF       : +1 Oct
